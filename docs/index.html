What is Gstrl?

Gesture + voice control for macOS. Pinch to move cursor. Circle to screenshot. Speak to run commands. All on-device via Apple Vision + SFSpeechRecognizer.

For when you're lying back, presenting, or just tired of reaching across your desk.

Highlights

  • Pinch → cursor — drag your cursor by pinching and moving
  • Circle → screenshot — draw a circle to capture that region to clipboard
  • Speak → commands — "press enter", "command z", or just dictate text
  • Swipe → arrows — open hand flick in any direction
  • Scroll → joystick — accelerates the longer you hold
  • Delete → escalating — hold 🤙: chars → words → lines → all
  • Liquid Glass island — floating status pill, macOS 26+ native
  • Configurable — FPS, cursor speed, scroll speed. Tweak to taste.

Install

git clone https://github.com/TomYang-TZ/Gstrl.git
cd Gstrl
make install

Requires macOS 14+, a webcam, and Swift 5.9+. Grant Camera, Accessibility, Screen Recording, and Speech Recognition permissions when prompted (app opens settings pages automatically).

When to use it

📸 Want to screenshot something — just circle it.

🎤 Presenting — swipe through slides from across the room.

🦾 RSI — alternate between mouse and gestures.

♿ Accessibility — another input channel beyond mouse/keyboard.
