## Codebase Patterns
- GestureAction enum lives in InputDispatch.swift — use .pressKey(UInt16) for simple keys, .pressModifiedKey for modifier combos
- Carbon.HIToolbox provides kVK_* key code constants (e.g. kVK_UpArrow = 126)
- Speech files go in Sources/iGest/Speech/
- SpeechEngine.onResult delivers cumulative transcription text (not deltas) — fires on background queue
- Key codes for letters: a=0, c=8, v=9, z=6
- SpeechController lives in Sources/iGest/Tracking/, wraps SpeechEngine with UI callbacks (onLabelUpdate)
- InputDispatch.perform() is a static method — call directly with a GestureAction
- SpeechController dispatches onResult handling to main thread to avoid race conditions with timers
- onLabelUpdate callback sets appState.gestureLabel which DynamicIslandView displays
- VoiceCommandResult.command includes displayName for UI feedback (e.g. "↓ Down Arrow")
